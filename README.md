# Expeerify

## Project status
Legacy personal proof-of-concept, originally created as an experimental integration between an interactive demo, REST/API communication, database-backed analysis, and report/result views.

## Why this repository exists
This repository is preserved as a public, non-confidential project that can be used as a modernization and AI-assisted analysis sandbox.

## Original goals
- Experiment with Unreal-to-REST communication.
- Store or process scenario/demo data.
- Generate report and result views.
- Explore web/API/database integration.

## Current modernization use
- AI-assisted architecture documentation.
- Accessibility review of generated report pages.
- Legacy code inventory extraction.
- Python helper scripts for analysis and reporting.
- Potential modernization of reporting UI.

## Tech areas
- Web UI
- API layer
- Database schema
- JavaScript/PHP-era implementation
- Unreal demo artifact
- HTML report output

## Not intended as
- A polished production system
- A current best-practice frontend implementation
- A maintained public product
